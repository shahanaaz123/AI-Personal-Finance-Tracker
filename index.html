# 💰 FinSmart AI — Student Finance Tracker
 
An AI-powered personal finance tracker built for smart student budgeting.
Built as a capstone project for Navigate Labs AI.
 
## Features
- 📊 Dashboard with spending charts
- 📋 Expense logging with category tags
- 🎯 Budget planner with sliders
- 🤖 AI Advisor (Navigate Labs API)
- 🐙 GitHub-integrated workflow
 
## Tech Stack
HTML · CSS · JavaScript · Chart.js · Navigate Labs AI API
 
## Setup
1. Clone: `git clone https://github.com/YOUR_USERNAME/ai-finance-tracker.git`
2. Open `index.html` in browser — no build step needed!
